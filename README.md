# Fundamental_Booster

Interactive Personal Data Collector:
A simple Python CLI program that collects personal information from the user and displays details about the entered data, including its type and memory address.

Features:
Collects user input (name, age, height, favourite number)
Demonstrates different Python data types (str, int, float)
Displays type and memory address using type() and id()
Calculates approximate birth year from the given age

Technologies:
Python 3
